# IdleDuelist

A turn-based combat game built with FastAPI and vanilla JavaScript.

## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Start the server:**
   ```bash
   start_server.bat
   ```
   Or manually:
   ```bash
   python start_server.py
   ```

3. **Open your browser:**
   Navigate to: http://localhost:8000
